# 🐱 Monsieur Poms - The Official Website

Welcome to the hilariously outdated official website of Monsieur Poms, the internet's most distinguished fat orange cat.

Built with [Hugo](https://gohugo.io/) - the world's fastest static site generator.

## Local Development

```bash
# Install Hugo
# On macOS: brew install hugo
# On Linux: snap install hugo or download from https://gohugo.io/

# Run local dev server
hugo server

# Build for production
hugo --minify
```

## Deployment

Automatically deploys to GitHub Pages on push to `main` branch.
